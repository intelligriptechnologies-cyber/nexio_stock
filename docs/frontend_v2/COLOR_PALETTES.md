# Color Palette & Concept Mocks for Barstock

We explored and generated color palettes on [Realtime Colors](https://realtimecolors.com) and [Coolors](https://coolors.co) to find combinations that suit a professional, high-volume retail POS and inventory management application like **Barstock**.

To visualize how the application looks under each theme, we created an interactive preview page:
- **[owner_dashboard_themes.html](file:///c:/barstock/docs/frontend_v2/owner_dashboard_themes.html)**: Open this in your browser to dynamically switch between all 7 themes on the actual Owner Dashboard layout.

Below are the details and snapshots for the top 7 candidates:

---

## Candidate 1: Professional & Stable (Slate Blue / Emerald / Clean Slate)
This palette is ideal for a stable, professional, and reliable corporate look. Deep slate blue provides structure, while emerald green provides a positive success/add status.

- **Primary**: `#1e3a8a` (Slate Blue) — Structure, main sidebars, primary branding.
- **Secondary**: `#475569` (Slate Gray) — Secondary actions, subtitles, borders.
- **Accent**: `#10b981` (Emerald Green) — Success, added/completed states, pay buttons.
- **Background**: `#f8fafc` (Clean Slate) — Low-glare, comfortable background.
- **Surface**: `#ffffff` (White) — Content cards, forms, tables.

## Candidate 2: Green Mana Theme (Obsidian Black / Green Mana / Light Slate)
The core requested Green Mana theme. Built on a rich midnight obsidian black foundation, featuring vibrant MTG-style Green Mana (`#22c55e`) as the central accent color for primary actions and active indicators.

- **Primary**: `#0b0f19` (Obsidian Black) — Main sidebars, primary structural layout, bold headers.
- **Secondary**: `#475569` (Slate Gray) — Border lines, secondary buttons, labels.
- **Accent**: `#22c55e` (Green Mana Green) — Primary actions (Checkout, Restock, Approve), status indicators, highlights.
- **Background**: `#f8fafc` (Low-glare Light Slate/Gray) — Soft layout background.
- **Surface**: `#ffffff` (White) — Table rows, cards, white containers.

## Candidate 3: Modern, Sleek & Premium (Deep Navy / Bright Teal / Minimalist White)

## Candidate 4: Electric Purple (Dark Mode / Deep Obsidian)
A premium dark mode concept. Employs deep obsidian and slate blue for surfaces to minimize eye strain in low-light environments (like bars or clubs), highlighted by a neon electric purple accent.

- **Primary**: `#030712` (Deep Obsidian) — Header, sidebar, and primary dark contrast backgrounds.
- **Secondary**: `#4b5563` (Dark Gray) — Borders and secondary options.
- **Accent**: `#8b5cf6` (Electric Purple) — Highlights, active navigation, active buttons.
- **Background**: `#0f172a` (Dark Slate Blue) — Ambient background.
- **Surface**: `#1e293b` (Slate Card Surface) — Dashboard cards and panels.
